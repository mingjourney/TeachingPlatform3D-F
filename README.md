# 前端学习 项目

这是一个3D可视化教学平台，将学习的技术聚合。

## 安装

首先，确保你已经安装了Node.js和Yarn。然后，按照以下步骤安装项目的依赖项：

1. 克隆项目到本地:

   ```bash
   git clone git@github.com:mingjourney/TeachingPlatform3D-F.git
   cd TeachingPlatform3D-F
   ```

2. 安装依赖
   ```bash
   yarn install
   ```

## 运行

你可以使用以下命令来运行项目：

- 本地开发模式：

  ```
  yarn dev
  ```

- 构建生产版本：

  ```
  yarn build
  ```

- 预觏生产版本：

  ```
  yarn preview
  ```

- 运行ESLint检查：

  ```
  yarn lint
  ```

4. 提交你的更改：

